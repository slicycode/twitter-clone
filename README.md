**Twitter-clone** is my second project which is a clone of the *twitter's* homepage. 
I made this project with **Next.js**.

You can clone it and use it on local :

```bash
yarn run dev
```
or
```bash
npm start
```
Make sure to build the project first before running it :

```bash
next build
```
